# Pesawat Kertas

## Project Overview
Pesawat Kertas is an innovative project aimed at exploring the principles of aerodynamics through the design and creation of paper airplanes. The project encompasses tutorials, flight tests, and design challenges, engaging users in a fun and educational manner.

## Features
- **Design Tools**: Interactive tools for designing your own paper airplane models.
- **Flight Test Log**: Record and analyze the flight tests of your paper airplanes.
- **Community Challenges**: Participate in design challenges and competitions with the community.
- **Educational Resources**: Access tutorials and articles on the science of flight and aerodynamics.
